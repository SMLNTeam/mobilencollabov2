# Setup

```sh
# 0. clone fork project
git clone https://github.com/Yourname/mobilencollabov2.git

# 1. virtual environment setup
python -m venv venv

# 2. venv run
# windows
source myvenv/Scripts/activate

# MacOS
source myvenv/bin/activate

# 3. runserver
python manage.py runserver
```
